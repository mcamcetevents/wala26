# TECH WALA 2K26 - Web Project

This is a complete, standalone web project for the **TECH WALA 2K26** Technical Symposium organized by the **Department of MCA, MCET**.

## 🚀 Version
- **Format**: Pure HTML5 / CSS3 / JavaScript (Vanilla)
- **Design System**: Electric Indigo & High-End Glassmorphism
- **Icons**: Lucide Icons (CDN)
- **Fonts**: Outfit & Inter (Google Fonts)

## 📁 Project Structure
- `index.html`: Main entry point with semantic structure.
- `css/style.css`: Modern, responsive CSS with theme variables and glassmorphism.
- `js/script.js`: Dynamic rendering logic for events, schedule, and modals.
- `google_apps_script.js`: Backend script for handling form submissions (if applicable).

## ✨ Features
- **Dynamic Theme**: Switch between professional Dark and Light modes.
- **Responsive Grid**: Interactive event cards that look great on all devices.
- **Animated Modals**: Clean, detailed view for every event's rules and prizes.
- **Interactive Schedule**: A visual timeline of the day's flow.
- **Google Integration**: Linked to official registration forms and Google Maps.

## 🛠️ How to Run
Simply open `index.html` in any modern web browser or use a "Live Server" extension in VS Code.
